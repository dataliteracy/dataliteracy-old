Unit 2 : Data Sources, Context, and Privacy

Data literacy can support Nepal government’s priorities by fostering an environment where citizens have the knowledge to use data for evidence-based decision making and to increase transparency and accountability in public service delivery.

Modules for Unit 2

  • Module 1

  • Module 2

Introduction to the world of data

Instructors: Sunnie Joshi, Craig Hammer


Topics covered:

  • Definition of data and open data
  • Explanation of data-driven decision making
  • Introduction to types of attributes:
    • Categorical
    • Numerical
    • Ordinal
  • Visualizing data using:
    • Dot plots
    • Pie charts
    • Bar charts
    • Stacked bar charts

The Data Pipeline

Instructors: Sunnie Joshi, Craig Hammer


Topics covered:

  • Common data formats
  • Ease of use
  • Introduction to open data formats:
    • CSV
    • XML
    • JSON
  • Examples of use with different tools
